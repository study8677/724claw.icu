## 🚀 快速开始

```bash
# 克隆 & 安装
git clone https://github.com/study8677/724claw.icu.git
cd 724claw.icu && npm install

# 开发模式
npm run dev

# 生产部署（使用 pm2）
npm run pm2:start
```
